Práctica N° 6

Crear una página HTML donde se muestre sus datos personales y los enlaces para cada mostrar la solución de los ejercicios propuestos utilizando el lenguaje JavaScript.

1) Crear la página de tal forma que cuando ingrese se muestre el mensaje “Bienvenido a las paginas dinámicas con JavaScript”.

Ejercicio 1

2) Realizar una página que solicite un número y verifique que la entrada esté comprendida entre 1 y 50

Ejercicio 2

3) Solicitar un número y generar su tabla de suma, resta, multiplicación y división.

Ejercicio 3

4) Crear un botón cuando se pulse se ella desplegar el mensaje hola a todos

Ejercicio 4

5) Crear tres cuadros con bordes de color rojo, azul y verde y que cuando pase el ratón por encima de ellos el fondo se cambie al color que tiene el borde y que cuando salga el fondo vuelva a ser blanco.

Ejercicio 5

6) Crear una página para realizar la conversión de bolivianos a dólares o viceversa, también debe tener un botón que indique limpiar campo

Ejercicio 6

7) Realizar una página que reciba un nombre y un botón que diga SALUDAR con la funcionalidad de que al pulsar se muestre una ventana saludando de acuerdo a la hora del día. Ej, Si es por la tarde deberá indica “Buenas tardes Marco”

Ejercicio 7

8) Leer una cadena de texto en un formulario y usando un botón mostrar el número de caracteres que se ha introducido e imprimir la cadena en orden inverso.

Ejercicio 8

9) Crear una página para simular el lanzamiento de un dado

Ejercicio 9

10) Realizar una página que implemente una calculadora aritmética

Ejercicio 10

11) Crear un rompecabezas de 4 x 4 en cada celda puede ir o un numero o una letra o parte de una imagen.

Ejercicio 11

12) Programar un juego usando JavaScript (Puede tomar como base alguno que ya existe y personalizarlo)
